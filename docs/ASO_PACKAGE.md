# ASO Package — Survival Trail: Frontier

## Naming rationale
- **Under the 30-char title limit** — "Survival Trail: Frontier" = 26 chars.
- **High-intent search terms** — *survival*, *trail*, *frontier* all in the title.
- **Avoids protected IP** — no "Oregon Trail." See `LEGAL_AND_NAMING.md`.
- **Reads as a real strategy/sim game** — genre-typical naming pattern (`Big Word: Modifier`).

## Google Trends comparison (run manually — Trends is not callable from this build env)
Trends reports **normalized search interest** — read as *relative direction, not absolute demand*.
Compare these five terms in the Trends UI (worldwide + top English markets, 12-month):
`survival game` · `strategy game` · `simulation game` · `frontier game` · `trail game`

Expected shape (validate, don't assume): *survival* and *strategy* dominate evergreen interest; *simulation* steady; *frontier* and *trail* are lower-volume but on-theme and lower-competition. → Title harvests the high-interest heads (survival/strategy/simulation in keyword field + subtitle) while the on-theme tail (frontier/trail) differentiates. Re-run before each major update; tune the **keyword field and subtitle**, not the locked title.
